# button-group

Используется для отображения группы однотипных кнопок.


## Примеры

```html
<div class="button-group">
  <button type="button" class="button button_theme_pale button_view_pseudo">Button</button>
  <button type="button" class="button button_theme_pale button_view_pseudo">Button</button>
  <button type="button" class="button button_theme_pale button_view_pseudo">Button</button>
</div>
```
